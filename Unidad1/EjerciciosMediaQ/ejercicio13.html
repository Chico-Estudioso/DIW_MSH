Ejercicios media queries

Ejercicio 1.

Modifica el color de fondo de body de amarillo a "lightblue" cuando la anchura del viewport sea de 600px o menos.

Ejercicio 2.

Oculta un elemento cuando la anchura del viewport sea de 600px o más.

Ejercicio 3.

Modifica el color de fondo de body a "pink" si la anchura del viewport es de 800px o mayor, a "lightgreen" si está entre 400px y 799px. Si es menor de 400px, el color de fondo será "lightblue".

Ejercicio 4.

Crea un menú de navegación adaptativo. Se mostrará horizontalmente para anchuras mayores de 600px y verticalmente para menores de 600px.

Ejercicio 5.

Crea un layout adaptativo como el siguiente, donde los dos breakpoints son a 600px y 992px.

Ejercicio 6.

Modifica el aspecto de un div añadiendo un padding, borde y color de fondo que resalte sólo para anchuras de pantalla entre 600 y 900px y para anchuras mayores de 1100px.
