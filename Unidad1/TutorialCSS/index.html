Explicación:

La propiedad CSS box-sizing determina cómo se calcula el tamaño total de un elemento HTML, incluyendo su contenido (content), relleno (padding) y borde (border).

Existen dos valores principales para box-sizing:

  • content-box: Con este valor, el tamaño total se calcula considerando únicamente el contenido interior del elemento, excluyendo el padding y el border.
  • border-box: Con este valor, el tamaño total se calcula incluyendo el contenido, el padding y el border, lo que hace que el tamaño total sea más predecible.
